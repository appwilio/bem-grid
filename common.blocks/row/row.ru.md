# Row

Контейнер для колонок. По смыслу схож с тэгом <tr> ;-)

## Использование блока

``` js
{
    block : 'row',
    content : [

    ]
}
```

## Модификаторы блока

### _reverse

Задает обратный порядок колонок.

### _no-flexbox

Будет автоматически добавлен блоку `row`, если браузер не поддерживает flexbox'ы, а затем
создаст деградацию до таблиц.

## Elements of a block

### __col

Колонка сетки.
